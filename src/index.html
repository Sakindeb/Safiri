Safely

This is a specialized low-level software tailored for Windows operating systems. This software serves multiple purposes, including recording active sessions, monitoring network activities, analyzing IP addresses connected to ports, scanning downloaded files for potential threats, and providing offline password storage. The software employs APIs from ipinfo and VirusTotal to enhance its capabilities and ensure comprehensive security measures. I Know a Spot

A dynamic web application designed to bring together individuals who have a passion for meat and provide them with a platform to discover and share their favorite meat-centric spots. Built using the MERN (MongoDB, Express, React, Node.js) stack, the app empowers users to not only explore existing meaty destinations but also contribute by adding their own recommended spots. The integration of the Google Maps API adds an interactive map feature for visualizing the locations and allows for easy geocoding of user-submitted addresses.
